Welcome to **Modern Halftones and Dithering**, a utility for making 1-bit, monochrome images from your color or grayscale images!

To get started, please download the PDF of the zine that serves as some background info and history on reproducing black and white images, as well as a manual for the program. [Get the zine here](PASTE_ZINE_LINK_HERE).

If you have a PC, you should download [this zip file](https://github.com/heckacharla/Modern-Halftones-and-Dithering/releases/download/v0.83/Modern_Halftones_for_WINDOWS_v086.zip), which has the .exe and everything you need to run. 
Note: Windows may be like "I am not sure if this is safe," but you can click "More Info" and then "Run anyway" to proceed. It's safe, Windows just gets nervous about software it hasn't seen before. (See FAQ below)

If you have a Mac, you should download [this DMG file](https://github.com/heckacharla/Modern-Halftones-and-Dithering/releases/download/v0.83/ModernHalftones.dmg), which has everything you need to run. When you open the DMG, you need to drag everything, including the .csv files and the .png, into a folder of your choice. (We recommend Applications -- just drag everything into that Application folder alias.) Keep everything together or the app won't work. The .csv files have to be in the same folder as the app. 
Note: MacOS may warn you when you try to open the app. If that happens, just right click and choose open from the menu, then "open" from the dialog box. (See FAQ below)

If you use Linux, or want to access the latest version of the source code for whatever reason, it's [right here](https://github.com/heckacharla/Modern-Halftones-and-Dithering/releases/download/v0.83/Modern_Halftones_Source_Code_v083.zip).
Note: No one warns you about anything if you use Linux. You do the warning.

The program and the source code is all created with the open source MIT License.

The zine has its own license, which is in its .zip file. 

Have fun!

FAQ:

**Why does the Mac/PC version get nervous about me running it?**
Both Apple and Microsoft are heavily and admirably focused on your security, and if someone is running a program/app/exe they want to be sure it's not going to do something awful. To signal an app is "safe" you can join the Apple Developer Program for $100 a year and "digitally sign" your apps so the OS won't freak out. With Microsoft you do something similar with a third party; it's about $70 - $100 a year. I love this app and I have devoted a lot of time to it, but it's explicitly free -- don't send me money, ever (send zines though!) -- and I have not joined either of those programs. I may in the future; if so, I'll update these builds asap!

If you're worried about the apps, you can always build from the source code. Learning to build the app from the source is both super easy and kind of a fun project. At the absolute most basic download Visual Studio Code, add the Python module, download the source, and point VSCode at a file in the source called Rastime.py. Run that. Boom.
