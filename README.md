Welcome to **Modern Halftones and Dithering**, a utility for making 1-bit, monochrome images from your color or grayscale images!

To get started, please download the PDF of the zine that serves as some background info and history on reproducing black and white images, as well as a manual for the program. Get the zine here.

If you have a PC, you should download this zip file, which has the .exe and everything you need to run.
If you have a Mac, you should download this zip file, which has everything you need to run.

If you use Linux, or want to access the latest version of the source code for whatever reason, it's right here.

The program and the source code is all created with the open source MIT License.

The zine has its own license, which is in it's .zip file.

Have fun!
